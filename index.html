Editor Shortcuts

  • W, A, S, D / Arrow Keys : Move Camera
  • Ctrl + Arrow Keys : Jump to Map Edge
  • Left Click / Drag : Place Block/Entity
  • Right Click / Drag : Erase (Remove FG first, then BG)
  • Middle Click : Copy Tool under cursor (Pipette)
  • V : Selection Tool (Drag to select)
  • X : Eraser Tool
  • Ctrl + Z : Undo
  • Ctrl + Y / Ctrl+Shift+Z : Redo
  • N / M : Zoom Out / Zoom In
  • Del / Backspace : Delete Selection
  • Esc : Exit Editor
